Versionamento de código é a ação de salvar todas as alterações feitas durante o desenvolvimento do código, processo importantíssimo para o desenvolvimento do projeto. Esse processo ajuda a organizar melhor os objetivos de cada parte do desenvolvimento do seu código e a dividir tarefas com colaboradores para a conclusão desses objetivos. Ao analisar constantemente as versões, é possível resgatar códigos antigos com eficiência e garantir que erros despercebidos possam danificar o projeto.
